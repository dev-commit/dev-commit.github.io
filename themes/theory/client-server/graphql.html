GraphQL

  • GraphQL - синтаксис, который описывает как запрашивать данные, и, в основном, используется клиентом для загрузки данных с сервера
    GraphQL имеет три основные характеристики
  1. Позволяет клиенту точно указать, какие данные ему нужны
  2. Облегчает агрегацию данных из нескольких источников
  3. Использует систему типов для описания данных
