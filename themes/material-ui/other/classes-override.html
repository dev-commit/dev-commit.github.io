Переопределение стилей из classes

Место использования компонента

<MyCheckbox className={classes.checkbox} /> checkbox: { padding: '10px', }

Базовый компонент

<Checkbox className={classes.root} /> root: { color: 'blue', }
  • В итоговом варианте, стили объеднятся
root: { padding: '10px', color: 'blue }
