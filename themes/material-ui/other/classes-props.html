Передача classes

  • Из родительного компонента пропом можно передать clasess в дочерний компонент. При этому у дочернего не должно быть стилей withStyles
  • Если у дочернего компонента будут свои стили, то будет предупреждение: Material-UI: the key wrapper provided to the classes prop is not implemented in Component. You can only override one of the following
<Component classes={classes} />
  • Если у дочернего компонента есть класс root, его можно переопределить из родительского, передав в пропе класс родительского компонента classes.paperRoot
<Component classes={{ root: classes.paperRoot }} />
