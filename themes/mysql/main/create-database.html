Создание базы данных


1. Кодировка: utf_general_ci / utf_unicode_ci
2. Первичный ключ: UNSIGNED – диапазон значений только положительные числа
(у INT в диапазон по умолчанию включены еще и отрицательные числа)
3. Типы данных:
	VARCHAR - малая длина записи (нужно указывать число, н-р: 500)
	LONGTEXT - максимальная длина записи
