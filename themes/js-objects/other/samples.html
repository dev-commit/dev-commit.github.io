Примеры

Сортировка элементов массива

  • Есть массив объектов. У объектов есть key. Нужно отсортировать элементы массива по key объектов по возрастанию
const arr = [ { key: 7, name: '' }, { key: 1, name: '' }, { key: 2, name: '' }, ]; arr.sort((a,z) => a.key - z.key); console.log(arr); // [ // { key: 1, name: '' }, // { key: 2, name: '' }, // { key: 7, name: '' }, // ];

Плоский массив из объекта

const checkedRows = { '0': [1,2,3], '1': [4,5,6], } const b = Object.values(checkedRows).flatMap(value => value); // => [1, 2, 3, 4, 5, 6]

Задание свойств объекта

const text = 'Some text'; function getId() { return 7; } let obj = { text, ['prop_' + getId()]: 'some prop' } console.log(obj); // Object {text: 'Some text', prop_7: 'Some prop'}

Метод для разных объектов

var speak = function() { console.log(this.sound, this.name); } var animal = { sound: 'Mrr', name: 'Cat', speak: speak } animal.speak(); // Mrr, Cat
