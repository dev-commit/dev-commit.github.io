Выравнивание элементов

У родительского элемента (горизонтальное) div { justify-items: stretch; - блоки растягиваются по ширине (по умолчанию) justify-items: start; - по левому краю. блоки не растягиваются (только по ширине контента) justify-items: end; - по правому краю. блоки не растягиваются (только по ширине контента) justify-items: center; - по центру. блоки не растягиваются (только по ширине контента) } У родительского элемента (вертикальное) div { align-items: stretch; - блоки растягиваются по высоте (по умолчанию) align-items: start; - сверху align-items: end; - снизу align-items: center; - по центру } У дочернего элемента (горизонтальное) div { justify-self: stretch; justify-self: start; justify-self: end; justify-self: center; } У дочернего элемента (вертикальное) div { align-self: stretch; align-self: start; align-self: end; align-self: center; }
