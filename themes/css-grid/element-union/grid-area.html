Обращение по Grid Area

Обращение к линиям по имени (создание Grid Area)

(результат аналогичен)

Имена выбираются провизвольно, но если приписывать к именам -start или -end будет создаваться Grid Area.

В .cards указали main-start и main-end для строк и столбцов, поэтому grid-area: main;


.cards {
	display: grid;
	grid-gap: 20px;
	grid-template-columns: [side-start] 1fr
                           [main-start] 1fr 1fr
                           [main-end];
	grid-template-rows:    [main-start] 200px 200px
                           [main-end];
}
.card:nth-child(1) {
	grid-area: main;
}

Обращение по Grid Area


.cards {
	display: grid;
	grid-gap: 20px;
	grid-template-columns: repeat(3, 1fr);
	grid-template-rows: 200px 200px;
	grid-template-areas: "header  header  menu"
						 "content content content";
}
.card:nth-child(1) {
	grid-area: header;
}
.card:nth-child(2) {
	grid-area: menu;
}
.card:nth-child(3) {
	grid-area: content;
}
Если нужно оставить область пустой, ставим "." 

div {
	grid-template-areas: "header header  menu"
					     "  .    content content";
}
