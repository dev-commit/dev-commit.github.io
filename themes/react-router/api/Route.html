Route

  • Route - отображение UI, следит чтобы совпадали пути между компонентом и адресной строкой
  • Route технически является компонентом, но не создаёт DOM-элемента
  • exact - компонент будет рендериться только по указанному адресу, а не по всем адресам, связанным с ним
<Route path='/' exact component=HomePage />
