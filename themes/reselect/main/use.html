Ипользование

Установка

npm i --save-dev reselect
  • Селектор - функция, принимающая state и возвращающая значение, полученное на основе state
  • reselectmapStateToProps срабатывает каждый раз, когда происходит изменение state

Использование

import { createSelector } from 'reselect'; const getUsers = (state) => { return state.users; } // getUsers - селектор, чтобы получить значения // users - придут полученнные значения из getUsers export const getUsersSelector = createSelector(getUsers, (users) => { return users.filter(u => true); }) const mapStateToProps = (state) => { return { users: getUsersSelector(state) } } const mySelector = createSelector( первыйСелектор, второйСелектор, (значениеПервогоСелектора, значениеВторогоСелектора) => { // } )
