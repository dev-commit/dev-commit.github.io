Миксины, примеси (@mixin)

Позволяют сделать блок стилей, который может быть использован повторно в любом месте документа 

@mixin large-text {
	font:{
		family: Arial;
		size: 30px;
		weight: bold;
	}
	color: red;
}
p {
	@include large-text;
}



p {
	font-family: Arial;
	font-size: 30px;
	font-weight: bold;
	color: red;
}
Миксины можно подключать друг в друга 

@mixin large-text {
	...
	@include small-text;
}
Миксины могут принимать аргументы, которые передаются при подключении миксина 

@mixin my-border($color, $width) {
	border:{
		color:$color;
		width:$width;
		style:dashed;
	}
}
p {
	@include my-border(red, 2px);
}



p {
	border-color: red;
	border-width: 2px;
	border-style: dashed;
}
Значение аргумента по умолчанию 

@mixin my-border($color, $width:10px) {
	border:{
		color:$color;
		width:$width;
		style:dashed;
	}
}
p {
	@include my-border(red);
}
