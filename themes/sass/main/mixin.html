Миксины, примеси (@mixin)

  • Миксины - позволяют сделать блок стилей, который может быть использован повторно в любом месте документа

Миксины можно подключать друг в друга

@mixin large-text { ... @include small-text; }

Миксины могут принимать аргументы, которые передаются при подключении миксина

Значение аргумента по умолчанию

@mixin my-border($color, $width:10px) { border:{ color:$color; width:$width; style:dashed; } } p { @include my-border(red); }
