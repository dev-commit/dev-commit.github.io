Контейнеры

.container # контейнер для фиксированной ширины .row # строки .col # колонки .container-fluid # контейнер для 100%-ной ширины .d-flex # display:flex
  • Цифры в наименовании классов колонок показывают, сколько колонок из 12-ти возможных в ряду вы бы хотели использовать. Так, если вы хотите использовать три колонки одной ширины, используйте .col-sm-4

Колонки

.col # колонка .col-sm # адаптивная колонка для sm(576px) и больше .col-4 # колонка, занимающая 4 ячейки из 12 в сетке
