Установка

Cypress Geting Started npm i --save-dev cypress

package.json

{ "scripts": { "cypress:open": "cypress open", // запуск тестов "cypress:run": "cypress run" // record screenshots, videos and failures }, "devDependencies": { "cypress": "^" } }

Запуск

npm run cypress:open npx cypress open // запуск тестов npx cypress run // record screenshots, videos and failures
