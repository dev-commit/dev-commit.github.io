Типы данных

# Примитивы 1. Number # Число 64-битный float 2. String # Строка 3. Boolean # Булевый, логический 4. null # Специальное пустое или неизвестное значение (присваивается самостоятельно) 5. undefined # Специальное неприсвоеное значение (возвращается из JS) 6. Symbol # Символ (ES6) 7. BigInt # Целые числа больше 253 BigInt(значение) # Объекты 8. Object # Array, Function, Object
  • Назначение типов данных: хранить разного рода информацию
  • Примитивам нельзя назначать методы (при считывании будет undefined), но считывать их можно
  • Специальные числовые значения Infinity и NaN

null и undefined

    undefined («неопределенный») представляет собой значение по умолчанию:
  • переменной, которой не было присвоено значения, т.е. объявленной, но не инициализированной переменной
  • функции, которая ничего не возвращает явно, например, console.log(1)
  • несуществующего свойства объекта
    • null («значение отсутствия значения») - значение, которое присваивается переменной явно

Преобразование типов

String(77); Number('77'); Boolean(77); 77 + ''; // string +'77'; // number !!'77'; // boolean 77.toString(); parseInt('77px'); parseFloat('77.2px')
