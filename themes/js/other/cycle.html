Циклы

Цикл for


for (var i = 1; i < 10; i++) {  
    console.log(i); 
}

Цикл while


var i = 0;  
while (i < 10) {  
    console.log(i); 
    i++;  
}

Цикл do while


var i = 0;  
do {  
    console.log(i); 
    i++;  
} while (i < 10);


break    // выход из цикла
continue // прекращает выполнение текущей итерации цикла и переходит на следующую
