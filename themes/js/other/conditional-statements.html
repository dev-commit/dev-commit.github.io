Условные операторы

Условный оператор case()

switch(Number(id)) { case 1: alert(id); break; case 2: alert(id); break; case 3: alert(id); break; default: alert("default"); } switch(name) { case 'name1': alert(name); break; case 'name2': alert(name); break; case 'name3': alert(name); break; default: alert("default"); } switch(name) { case 'name1': case 'name2': case 'name3': return false; }

Условный оператор if()

if (year < 2011) { alert('Это слишком рано..'); } else if (year > 2011) { alert('Это поздновато..'); } else { alert('Да, точно в этом году!'); }
