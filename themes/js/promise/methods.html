Методы

Конструктор промиса Асинхронная функция с коллбэками resolve и reject
resolve(result) - Функция вызывается при успешном выполнении промиса. Передается аргументом в .then
reject(error) - Функция вызывается при неудачном выполнении промиса. Передается аргументом в .catch или вторым аргументом в .then
Выполнение действий после выполнения промиса. Обработчик событий промиса onFulfilled и onRejected promise(result) -> функция или код при успешном выполнении
Выполняется обработчик onRejected promise(error) -> функция или код при неуспешном выполнении
Вызывается в любом случае, независимо была ли ошибка. Дает возможность запустить один раз определенный участок кода, который должен выполниться вне зависимости от того, с каким результатом выполнился Promise promise
Возвращает промис, который будет выполнен, после выполнения остальных промисов Массив промисов Новый промис, который резолвится, когда резолвятся все промисы из списка
Возвращает только первый успешно выполнившийся промис из списка Массив промисов Первый промис из списка
Создаёт успешно выполнившийся промис с результатом value value -> результат
Создаёт уже выполнившийся промис с ошибкой error value -> ошибка
