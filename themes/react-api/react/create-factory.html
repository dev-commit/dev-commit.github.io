React.createFactory()

  • Этот вспомогательный метод считается устаревшим, и мы рекомендуем использовать либо JSX, либо напрямую React.createElement()
  • Обычно вы не будете вызывать React.createFactory() напрямую, если вы используете JSX
Возвращает функцию, которая создаёт элементы React заданного типа аргументом type может быть строка содержащая имя тега (например, 'div' или 'span'), React-компонент (класс или функция) или React-фрагмент Функция, которая создаёт элементы React заданного типа
