Хук useImperativeHandle

Настраивает значение экземпляра, которое предоставляется родительским компонентам при использовании ref. В большинстве случаев следует избегать императивного кода, использующего ссылки. useImperativeHandle должен использоваться с forwardRef ref createHandle [deps]
eImperativeHandle(ref, createHandle, [deps])
