Регистрация компонента

Глобальная регистрация, работает для всех Vue объектов


Vue.component('component-a', {
	template: 'content'
});

Локальная регистрация


var app = new Vue({
	el: '#app',
	components: { 
        'component-a': {
            template: 'content'
        } 
    }
});

Локальная регистрация из объекта


var ComponentA = {
	template: 'content'
}

new Vue({
	el: '#app',
	components: {
		'component-a': ComponentA
	}
})
