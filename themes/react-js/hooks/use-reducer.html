Хук useReducer


const [state, dispatch] = useReducer(reducer, initialArg, init);
  • Альтернатива для useState. Принимает редюсер типа (state, action) => newState и возвращает текущее состояние в паре с методом dispatch
  • useReducer - даёт возможность управлять внутренним состоянием более сложного компонента с помощью редюсера

function Todos() {
	const [todos, dispatch] = useReducer(todosReducer);
	// ...
}
