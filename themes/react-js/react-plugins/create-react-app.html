Create React App

Getting Started Create React App Available Scripts
  • Create React App - создание одностраничных приложений, предоставляет команды для сборки фронтенда
  • «Под капотом» используются Babel и Webpack, eslint и т.д.

npx create-react-app my-app # пакеты
cd my-app                   # выбор директирии
npm start                   # start
npm run build               # создаст оптимизированную сборку приложения в папке build

npx create-react-app my-app --typescript # установка typescript
