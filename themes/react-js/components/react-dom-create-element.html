React.createElement

  • Код, написанный с использованием JSX, будет преобразован в React.createElement()
Создаёт и возвращает новый React-элемент определённого типа строка, содержащая:
  • Имя тега (например, 'div' или 'span')
  • React-компонент (класс или функция)
  • React-фрагмент
 props children Новый React-элемент определённого типа
