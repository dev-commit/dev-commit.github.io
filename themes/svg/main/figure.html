Полигон (polygon)

Путь (path)

    Доступные команды:
  • M = moveto
  • L = lineto
  • H = horizontal lineto
  • V = vertical lineto
  • C = curveto
  • S = smooth curveto
  • Q = quadratic Bézier curve
  • T = smooth quadratic Bézier curveto
  • A = elliptical Arc
  • Z = closepath
  • Атрибут preserveAspectRatio="none" позволяет поместить фигуру целиком в viewBox
  • При изменении размера блока svg размер фигуры будет меняться
