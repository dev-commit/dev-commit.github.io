Отмена изменений

# отменить все изменения # убрать из индексации все файлы и директории, # которые возможно уже были проиндексировалы с помощью git add git reset --hard HEAD git checkout -- file.txt # Отмена изменений файла git reset HEAD file.txt # Отмена индексации файла git commit --amend # Изменение последнего коммита # Все три команды вместе дают один коммит - второй коммит заменяет результат первого git commit -m 'initial commit' git add forgotten_file git commit --amend git checkout . # восстановить все git checkout "код коммита" # вернуть до состояния этого коммита # Просмотреть, а затем удалить непроиндексированные файлы и папки: git clean -nd git clean -fd # Просмотреть, а затем удалить ВСЕ непроиндексированные файлы и папки, включая игнорируемые Git-ом git clean -ndx git clean -fdx
