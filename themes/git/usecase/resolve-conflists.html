Разрешение конфликтов разницы "develop" и "feature"

git checkout develop # перейти в develop, если в ней есть обновления git pull # обновить ветку develop git checkout feature # перейти в feature git merge develop # merge из develop в свою ветку. ветка feature станет актуальной # появятся конфликты в файлах, поправить конфликты
