Отмена изменений. Изменяет указатель HEAD и, опционально, состояние индекса 

git reset
git reset --hard d8578edf8458ce06fbc5bb76a58c5ca4a58c5ca4 # откатиться к конкретному коммиту и удалить последующие (хэш смотрим в «git log»)
git reset HEAD имя_файла                                  # если изменения были индексированны и нужно откатиться перед самим коммитом
