CSS Media Queries

Варианты

/* Desktop First */ @media screen and (max-width: 480px) { /* Ширина экрана стала меньше 480px */ } /* Mobile First */ @media screen and (min-width: 480px) { /* Ширина экрана больше 480px */ } /* Несколько условий */ @media screen and (min-width: 600px) and (max-width: 980px) { /* Экраны от 600px до 980px */ }
