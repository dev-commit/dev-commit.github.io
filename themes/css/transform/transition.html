Transition

transition (анимированный переход)

transition: all 0s ease 0s; 1. transition-property # какие свойства анимируются 2. transition-duration # длина анимации 3. transition-timing-function # как вычисляются промежуточные состояния 4. transition-delay # время ожидания перед запуском эффекта div { transition: all 0.7s ease-in 0s; } div { background: orange; transition: all 0.7s ease-in 0s; } div:hover { background: blue; }
