Box-Shadow, Text-Shadow

Тень для блока 

div {
	box-shadow: 1px 2px 3px 4px #000000;
	box-shadow: inset 1px 2px 3px 4px #000000;
}
/* 1px - смещение по горизонтали */
/* 2px - смещени по вертикали */
/* 3px - размытие */
/* 4px - толщина */
Тень для текста 

div {
	text-shadow: 1px 2px 3px #000;
}
/* 1px - смещение по горизонтали */
/* 2px - смещени по вертикали */
/* 3px - размытие */
