Единицы измерения

px # абсолютная единица измерения % # относительная, рассчитывается относительно родителя vw # viewport width, относительная, рассчитывается относительно viewport vh # viewport height, относительная, рассчитывается относительно viewport em # относительная, рассчитывается относительно родителя (берет значение font-size) rem # root em, относительная, рассчитывается относительно html (берет значение font-size)
