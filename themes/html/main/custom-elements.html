Custom Elements

1. Normalize its attributes in your CSS Stylesheet (think css reset) 

stack {
	display: block;
	margin: 0;
	padding: 0;
	border: 0;
	...
}
2. Make it work in old versions of Internet Explorer by appending this script to the head (Important!) 

 <!--[if lt IE 9]>
 <script> document.createElement("stack"); </script>
 <![endif]-->
3. Used 

<stack>Overflow</stack>
4. Set attributes 

<stack id="st2" class="nice">Overflow</stack>
