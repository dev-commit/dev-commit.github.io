Таблицы

Структура HTML


<table> – контейнер
<tr> - table row (строка)
<td> - table data (ячейка)
<th> - table header (ячейка заголовка)
<td rowspan="2"> – объединение строк
<td colspan="2"> – объединение столбцов

Стилизация на html


<table
	style="border-collapse: collapse"
	border=1
	borderColor=#000000
	cellpadding=4
	cellspacing=0
	width="100%">
</table>


<table>
	<tr>
		<td colspan="2">1 столбец 1 строка</td>
		<!-- <td>2 столбец 1 строка</td> --> <!-- colspan="2" -->
		<td>3 столбец 1 строка</td>
		<td>4 столбец 1 строка</td>
	</tr>
	<tr>
		<td rowspan="2">1 столбец 2 строка</td>
		<td>2 столбец 2 строка</td>
		<td>3 столбец 2 строка</td>
		<td>4 столбец 2 строка</td>
	</tr>
	<tr>
		<!-- <td>1 столбец 3 строка</td> --> <!-- rowspan="2" -->
		<td>2 столбец 3 строка</td>
		<td>3 столбец 3 строка</td>
		<td>4 столбец 3 строка</td>
	</tr>
</table>
