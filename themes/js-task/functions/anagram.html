Анаграмма

  • Анаграмма - перестановка букв или звуков определённого слова, что в результате даёт другое слово или словосочетание

Постановка задачи

var input = [ "вертикаль", "кильватер", "апельсин", "спаниель", "австралопитек", "ватерполистка", "кластер", "сталкер", "стрелка" ]; // Результат: [ ["вертикаль", "кильватер"], ["апельсин", "спаниель"], ["австралопитек", "ватерполистка"], ["кластер", "сталкер", "стрелка"] ]
