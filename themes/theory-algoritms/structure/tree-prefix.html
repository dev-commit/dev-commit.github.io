Префиксное дерево

  • Префиксное (нагруженное) дерево - разновидность дерева поиска. Оно хранит данные в метках, каждая из которых представляет собой узел на дереве
  • Такие структуры часто используют, чтобы хранить слова и выполнять быстрый поиск по ним - например, для функции автозаполнения
  • Каждый узел в языковом префиксном дереве содержит одну букву слова. Чтобы составить слово, нужно следовать по ветвям дерева, проходя по одной букве за раз. Дерево начинает ветвиться, когда порядок букв отличается от других имеющихся в нем слов или когда слово заканчивается. Каждый узел содержит букву (данные) и булево значение, которое указывает, является ли он последним в слове
  • При составлении слов нужно начинать с корневого узла и спускаться вниз. Это дерево содержит слова: ball, bat, doll, do, dork, dorm, send, sense
