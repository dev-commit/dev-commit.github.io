Queue (Очередь)

  • Queue (Очередь) - абстрактный тип данных с доступом к элементам, организованный по принципу FIFO
  • FIFO (First In First Out) - «первый пришёл — первый вышел». Удалить элемент можно только после того, как были убраны все ранее добавленные элементы
  • Добавление элемента (enqueue — поставить в очередь) возможно лишь в конец очереди, выборка - только из начала очереди (dequeue - убрать из очереди), при этом выбранный элемент удаляется из очереди
  • Можно представить как очередь в продуктовом магазине: первым обслуживают того, кто пришёл в самом начале

Операции

  • enqueue - добавлять элементы в конец очереди
  • dequeue - удалять первый элемент из очереди
