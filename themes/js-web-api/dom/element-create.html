Добавление и удаление узлов

    Создание узлов:

  • document.createElement - создает новый элемент с указанным тегом
    • tag - тег для узла
  • document.createTextNode - создает новый текстовый узел с заданным текстом
    • text - текст для узла
  • elem.cloneNode - клонирование узла elem
    • bool - true - создаст «глубокую» копию элемента вместе с атрибутами, включая подэлементы; false - копия будет создана без дочерних элементов

    Вставка и удаление узлов:

  • parentElem.appendChild - добавляет elem в конец дочерних элементов parentElem
    • elem - добавляемый элемент, созданный через createElement
    • return elem - вставленный узел
  • parentElem.insertBefore - вставляет elem в коллекцию детей parentElem, перед элементом nextSibling
    • elem - добавляемый элемент, созданный через createElement
    • nextSibling - элемент, перед которым производится вставка. Если null - сработает как appendChild
    • return elem - вставленный узел
  • parentElem.removeChild - удаляет elem из списка детей parentElemт
    • elem - элемент для удаления
    • return elem - удаленный узел
  • parentElem.replaceChild - среди детей parentElem удаляет elem и вставляет на его место newElem
    • newElem - элемент для замены
    • elem - элемент для удаления
    • return elem - удаленный узел
  • elem.remove - удаляет элемент напрямую, не требуя ссылки на родителят

Создание

var element = document.createElement('div'); element.innerHTML = 'Go to google'; element.className = 'class-google';

Вставка

var wrapper = document.querySelector('#wrapper'); var element = document.createElement('div'); wrapper.appendChild(element); // внутри wrapper после контента wrapper.insertBefore(element, wrapper.firstChild); // внутри wrapper перед контентом wrapper.parentNode.insertBefore(element, wrapper); // перед wrapper
  • Если вы хотите переместить элемент на новое место – не нужно его удалять со старого
  • Все методы вставки автоматически удаляют вставляемый элемент со старого места

Замена

wrapper.parentNode.replaceChild(element, wrapper); // заменить элемент другим элементом

Удаление

wrapper.parentNode.removeChild(element); // удаление нового элемента wrapper.parentNode.removeChild(wrapper); // удаление элемента обертки document.body.removeChild(element); // удаление нового элемента (если родитель body)

Текстовые узлы для вставки текста

var div = document.createElement('div'); div.appendChild(document.createTextNode('text'));
