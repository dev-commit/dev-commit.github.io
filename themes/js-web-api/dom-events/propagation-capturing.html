1. Фаза погружения

  • Навешиваемые обработчики через <event>-свойство или через HTML-атрибуты, или через addEventListener(event, handler) с двумя аргументами, ничего не знают о фазе погружения, а работают только на 2-ой и 3-ей фазах
  • Чтобы поймать событие на стадии погружения, нужно использовать третий аргумент addEventListener: capture или значение true
  • Если аргумент false (по умолчанию), то событие будет поймано при всплытии
  • Если аргумент true, то событие будет перехвачено при погружении
elem.addEventListener('click', () => alert('Привет'), {capture: true}) elem.addEventListener('click', () => alert('Привет'), true)
    Чтобы убрать обработчик removeEventListener, нужна та же фаза
  • Если мы добавили обработчик addEventListener(..., true), то мы должны передать то же значение аргумента capture в removeEventListener(..., true), когда снимаем обработчик
