Web Forms

The <input> element in HTML4

text # поле с текстом password # поле с паролем checkbox # флажок radio # переключатель select # выпадающий список hidden # скрытое поле textarea # текстовое поле submit # кнопка для отправки формы file # поле для отправки файла image # кнопка с изображением button # кнопка

The <input> element in HTML5 (Web Forms 2.0)

number # поле с номером (можно использовать '.', ',') email # email url # url range # ползунок datetime # дата и время datetime-local # дата и время date # дата month # месяц week # неделя time # время

The <input> element in HTML5 (Web Forms ...)

tel # телефон search # поиск pattern # регулярные выражения color # выбор цвета reset # кнопка для очистки формы

Web Forms Attributes

min # number, ... max # number, ... readonly # text, ... checked # radio, checkbox selected # option placeholder # text disabled # all autofocus # text

The <output> element

  • <output> определяет область, в которую выводится информация, преимущественно с помощью скриптов
<form oninput="result.value=cm.value"> <input type="text" name="cm" autofocus> <output name="result">0</output> </form>
