Хуки

  • beforeEach - хук, который будет вызываться ПЕРЕД каждым тест-кейсом
  • afterEach - хук, который будет вызываться ПОСЛЕ каждого тест-кейса
  • beforeAll - хук, запустится 1 раз ПЕРЕД тем как запустится первый тест
  • afterAll - хук, запустится 1 раз ПОСЛЕ всех тест-кейсов
beforeEach(() => { // })
  • Если есть асинхронный код в beforeEach, нужно возвращать Promise
