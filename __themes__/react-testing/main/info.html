Snapshots

    Применять
  • Компонент максимально статичен и почи никогда не меняется (верстка, stateless компоненты)
  • Детерминированные компоненты (запуск одних и тех же тестов для компонента, который не меняется, должен возврещать одинаковые результаты)
  • При рефакторинге существующих компонентов (изменение только функционала без изменения вёрстки)
    • Не применять
  • Во всех остальных случаях
