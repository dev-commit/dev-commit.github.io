Обработчики событий

// обработчик события щелчка мыши $('.item').on('click', function(){ // }); // обработчик события изменения значения input $('input').on('change', function(){ // }); // выполняется после загрузки всех элементов (если скрипт подключен в head) $(document).ready(function(){ // обработчик события щелчка мыши $('.item').on('click', function(){ // id выбранного элемента const id = this.id; }); });

Удаление обработчиков событий

Метод .off() удаляет обработчики событий, назначенные методом .on $(document).off('scroll');

Делегирование обработчиков событий

Применяется при необходимости добавить обработчик события на динамически добавленный элемент

<button>Click me</button> <div class="container"></div> $('button').on('click',function(){ $('.container').append('<div class="item" id="item-1"></div>'); $('.container').append('<div class="item" id="item-2"></div>'); }); $('.container').on('click', '.item', function(){ console.log(this.id); });
