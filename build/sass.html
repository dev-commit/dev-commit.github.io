Переменные

Вложенность переменных

  • Переменная доступна только на текущем уровне вложенности, чтобы была видна везде, дописать !global
$bg-color: green !global;

Переменные в calc()

  • Необходимо записывать #{$spacing}
$spacing: rem(6px); .item { width: calc(50% - #{$spacing}); height: calc(50% - #{$spacing / 2}); }

Операции над цветами

lighten($color,10%) # осветляет исходный цвет darken($color,10%) # затемняет исходный цвет saturate($color,70%) # увеличивает насыщенность цвета от исходного desaturate($color,70%) # уменьшает насыщенность цвета от исходного transparentize($color,0.2) # добавляет к исходному цвету альфа-канал opacify($color, 0.1) # задание менее прозрачного цвета transparentize($color, 0.4) # задание более прозрачного цвета rgba(#333, 0.75) # перевод из hex в rgba

Миксины, примеси (@mixin)

  • Миксины - позволяют сделать блок стилей, который может быть использован повторно в любом месте документа

Миксины можно подключать друг в друга

@mixin large-text { ... @include small-text; }

Миксины могут принимать аргументы, которые передаются при подключении миксина

Значение аргумента по умолчанию

@mixin my-border($color, $width:10px) { border:{ color:$color; width:$width; style:dashed; } } p { @include my-border(red); }

Импорт других SASS файлов (@import)

Сконвертируется в css как @import если

  • Расширение css
  • Media-параметры
  • Присутствует http
  • Присутствует url

Импортируется в sass файл если

1. Расширение sass @import "main.sass"; 2. Расширение scss @import "main.scss"; 3. Файлы css (main.css) @import "main"; 4. Файлы-фрагмены (_main.sass / _main.scss) @import "main"; @import "example.scss"; @import "example1", "example2";

Директива @each

  • Если требуемся пройтись по списку значений
  • Устанавливает переменную в каждое из значений списка или карты и выводит содержащиеся в ней стили

Директива @at-root

  • Изымает одно или несколько правил из родительского селектора в корневой уровень документа. Можно использовать с селекторами первого уровня вложенности
  • Поднимает содержимые в ней правила в корень, отменяя цепочку от родителя
