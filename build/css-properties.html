Значение свойств

  • none - Отменяет заданные значения для свойства
  • unset - Комбинация ключевых слов initial и inherit. Ключевое слово unset устанавливает значение свойства как inherit, если свойство наследуется от своего родителя, в противном случае значение устанавливается как initial
  • inherit - Сообщает, что необходимо наследовать значение свойства у родительского элемента. Результат будет заметен только в том случае, если у родителя указанное свойство установлено
  • initial - Применяется для установки исходного значения свойства. Может пригодиться в нескольких случаях, к примеру, восстановить значения свойств, заданных браузером по умолчанию или задать начальное значение свойства, измененное в результате наследования

Overflow

Свойство overflow управляет отображением содержания блочного элемента, если оно целиком не помещается и выходит за область заданных размеров. div { overflow: auto; - добавление полос прокрутки, если контент не помешается в блок overflow: hidden; - обрезка контента, если контент не помешается в блок }

Background

div { background-image: url(img.png); # путь к изображению background-repeat: no-repeat; # повторение изображения background-position: center center; # позиционирование изображения (задние в px и %) background-size: contain; # изображение полностью помещается в блок background-size: cover; # изображение заполняет весь блок background-size: 50%; # задание в % background-color: #3b3b3b; # цвет фона background-attachment: fixed; # будет ли прокручиваться фоновое изображение background-origin: border-box; # устанавливает область расположения фона background-clip: border-box; # определяет как цвет фона или фоновое изображение будут выводиться под границами блока }

Сокращенная запись

div { background: color url no-repeat center center / cover; background: url(img.png) no-repeat center 50%; background: url no-repeat center; }

Задание нескольких фонов

div { background: url(img.png) no-repeat, #fff url(img.png) no-repeat 100% 0; }

attr()

  • Эта функция позволяет нам получить доступ к HTML, получить содержимое атрибута и передать его свойству содержимого CSS
  • Вы часто увидите использование attr() в таблицах стилей печати, где он используется для отображения URL-адреса ссылки после ее текста
  • Еще одно применение этой функции - использование для отображения альтернативного описания изображения, если оно не загружается
<div data-example="foo"></div> div { content: attr(data-example); }

image-set()

  • Функция позволяет указать список изображений, которые браузер сможет выбрать в качестве фонового изображения, основываясь на том, что он знает о возможностях своего дисплея и скорости соединения
  • Это аналогично тому, что вы делаете со свойством srcset
.responsive-background { background-image: image-set( "image.png" 1x, "image-2x.png" 2x, "image-print.png" 600dpi ); }

::slotted

  • Используемый для выбора элементов, помещенных в слот внутри шаблона HTML. ::slotted()
  • Предназначен для использования при работе с веб-компонентами, которые представляют собой пользовательские элементы HTML, определенные разработчиком
