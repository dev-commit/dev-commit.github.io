Семантика

  • Семантическая верстка - подбор тегов в соответствии со спецификацией
    Разметка
  • <section/> - смысловой раздел документа. Неотделяемый, в отличие от <article/>
  • <article/> - независимая, отделяемая смысловая единица, например комментарий, твит, статья
  • <aside/> - второстепенный контент
    • Теги
  • <strong/> - нечто важное
  • <b/> - выделение текста для привлечения внимания, без придания важности
  • <li/> - однородные элементы
  • <p/> - группировка фразовых элементов
    • Изображения
  • <img/> - содержательные изображения
  • <div/> -> css:background-image - фоновые изображения

HTML Content Model

  • ✖ Metadata ✖ Flow content - ПОТОКОВЫЙ КОНТЕНТ - блочные элементы (HTML4), без Phrasing Content
    • • Heading - заголовки
    • • Sectioning - секции
    • • Phrasing Content - ФРАЗОВЫЙ КОНТЕНТ - строчные элементы (HTML4)
      • • Embedded Content - встраиваемый контент: audio, video, img, canvas. Не является частью html
    • • Interactive - интерактивные элементы: button, a, label
  • ✖ Transparent - прозрачная модель, когда модель содержания элемента такая же как у родительского тега

Web Forms

The <input> element in HTML4

text # поле с текстом password # поле с паролем checkbox # флажок radio # переключатель select # выпадающий список hidden # скрытое поле textarea # текстовое поле submit # кнопка для отправки формы file # поле для отправки файла image # кнопка с изображением button # кнопка

The <input> element in HTML5 (Web Forms 2.0)

number # поле с номером (можно использовать '.', ',') email # email url # url range # ползунок datetime # дата и время datetime-local # дата и время date # дата month # месяц week # неделя time # время

The <input> element in HTML5 (Web Forms ...)

tel # телефон search # поиск pattern # регулярные выражения color # выбор цвета reset # кнопка для очистки формы

Web Forms Attributes

min # number, ... max # number, ... readonly # text, ... checked # radio, checkbox selected # option placeholder # text disabled # all autofocus # text

The <output> element

  • <output> определяет область, в которую выводится информация, преимущественно с помощью скриптов
<form oninput="result.value=cm.value"> <input type="text" name="cm" autofocus> <output name="result">0</output> </form>

Таблицы

Структура HTML

<table> – контейнер <tr> - table row (строка) <td> - table data (ячейка) <th> - table header (ячейка заголовка) <td rowspan="2"> – объединение строк <td colspan="2"> – объединение столбцов
