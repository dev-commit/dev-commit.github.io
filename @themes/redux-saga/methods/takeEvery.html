takeEvery

  • takeEvery
  • Используется когда необходимы результаты всех предыдущих задач
  • В этом случае все предыдущие саги и их возвращаемые значения являются взаимозависимыми, а не только самые последние
  • Очень похожа на redux-thunk по своему поведению и методологии. По сути, это оболочка yield take
  • takeEvery позволяет обрабатывать одновременные действия. Например, пользователь нажимает кнопку «Загрузить пользователя» 2 раза подряд с высокой скоростью, второй щелчок отправит действие USER_REQUESTED, в то время как fetchUser, запущенный при первом нажатии, еще не завершился
import { takeEvery } from 'redux-saga/effects' yield takeEvery(onAxiosResponse.type, checkAppCompatibilityBreakingFeatures)
