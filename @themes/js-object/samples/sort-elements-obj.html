Сортировка элементов массива

  • Есть массив объектов. У объектов есть key. Нужно отсортировать элементы массива по key объектов по возрастанию
const arr = [ { key: 7, name: '' }, { key: 1, name: '' }, { key: 2, name: '' }, ]; arr.sort((a,z) => a.key - z.key); console.log(arr); // [ // { key: 1, name: '' }, // { key: 2, name: '' }, // { key: 7, name: '' }, // ];
