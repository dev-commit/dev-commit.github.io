Перебор объекта

Цикл for in

Проходит через перечисляемые свойства объекта и его цепочку прототипов. Он пройдёт по каждому отдельному элементу. Порядок перебора соответствует порядку объявления для нечисловых ключей, а числовые или числовые строки сортируются Другое (очередное) имя свойства назначается переменной на каждой итерации Объект, по чьим свойствам мы проходим
for(let key in obj) { console.log(key); // ключи console.log(obj[key]); // значения }

Object.keys и forEach

const values = { name: 'Tony', age: 25 } Object.keys(values).forEach(key => { const data = values[key]; console.log(data); });

Перечисление всех свойств объекта

// возвращает массив со всеми собственными именами перечисляемых свойств объекта // (те, что в цепочке прототипов, не войдут в массив) const a = Object.keys(obj); // количество свойств const a = Object.keys(obj).length; // возвращает массив содержащий все имена своих свойств (перечисляемых и неперечисляемых) объекта const a = Object.getOwnPropertyNames(obj);
