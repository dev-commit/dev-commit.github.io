Проверка наличия свойства

  • Можно обратиться к любому свойству объекта, даже если его нет. Если свойство не существует, вернется undefined
Проверка наличия свойства 'name' в объекте // не различает отсутствующие свойства и те что были установлен в undefined const a = obj.name; // различает отсутствующие свойства и те что были установлен в undefined const a = 'name' in obj; // не берет наследованные свойства const a = obj.hasOwnProperty('name');
