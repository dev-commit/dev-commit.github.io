Кеширование

  • В API будет отправлен всего 1 запрос, т.к. они используют один и тот же ключ SWR - useUser, а запрос автоматически выводится, кешируется и распределяется
const App = () => { return ( <> <Content /> <Avatar /> </> ) } const Content = () => { const { user, isLoading } = useUser() if (isLoading) return <Spinner /> return <h1>С возвращением, {user.name}</h1> } const Avatar = () => { const { user, isLoading } = useUser() if (isLoading) return <Spinner /> return <img src={user.avatar} alt={user.name} /> }
