startsWith()

MDN
Метод помогает определить, начинается ли строка с символов указанных в скобках, возвращая, соответственно, true или false символы, искомые в начале данной строки позиция в строке, с которой начинать поиск searchString; по умолчанию 0 true или false

Пример

const str = "Быть или не быть, вот в чём вопрос."; console.log(str.startsWith("Быть")); // true console.log(str.startsWith("не быть")); // false console.log(str.startsWith("не быть", 9)); // true
