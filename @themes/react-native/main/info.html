Данные

Информация не проверенная
  • JS ни во что не компилируется. Он через "мост" (в новой архитектуре через турбо модули, проще говоря - это прослойка на С++) общается с нативными модулями платформы, JS строит дерево, рассчитываются стили и передаются через мост на платформу, которая уже используя свои View и отрисовывает это дерево
  • С ивентами то же самое. Событие нажатия произошло, эта инфа передается на сторону JS, JS выполняет свою логику и через турбо модули может дергать поведение нативных модулей платформы (открой мне камеру и т.д.)
