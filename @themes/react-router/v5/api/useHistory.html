useHistory()

Программная навигация по истории

Обект history

const history = useHistory(); history.goBack(); // перейти на предыдущую страницу history.push("/home"); // перейти на страницу "home"
  • location (object) - Текущее Location
    • pathname (string) - URL
    • search (string) - Query-параметры из URL
    • hash (string) - Хэш из URL
    • state (object) - Состояние, зависящее от местоположения, которое было предоставлено, например. push(path, state), когда это местоположение было помещено в стек. Доступно только в истории браузера и памяти
  • length (number) - Кол-во записей в History Stack
  • action (string) - Текущий Action (PUSH / REPLACE / POP)
    • PUSH - Переход с другой страницы
    • REPLACE - Открытие в новом окне
    • POP - ?
  • push(path, [state]) (function) - Помещает новую запись в History Stack
  • replace(path, [state]) (function) - Заменяет текущую запись на History Stack
  • go(n) (function) - Перемещает указатель в стеке истории на n записей
  • goBack() (function) - Аналогично go(-1)
  • goForward() (function) - Аналогично go(1)
  • block(prompt) (function) - Запрет навигации

Пример

import { useHistory } from 'react-router-dom'; const App = () => { const history = useHistory(); return <button onClick={history.goBack()}>Back</button> }
