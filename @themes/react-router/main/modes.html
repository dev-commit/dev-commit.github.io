Режимы работы

Режимы

Declarative Mode

Первоначальный формат
  • Декларативный режим обеспечивает базовые функции маршрутизации, такие как сопоставление URL-адресов с компонентами, навигация по приложению и предоставление активных состояний с помощью API

Data Mode

Новый формат
  • Перенеся настройку маршрутов за пределы рендеринга React, Data Mode добавляет загрузку данных, действия, состояния ожидания и многое другое

Framework Mode

Больше подходит для SSR
  • Framework Mode использует плагин Vite для расширения функциональности React Router включая следующие возможности: стратегии SPA, SSR и статического рендеринга, типобезопасный href и API модуля маршрутизации и т.д.
