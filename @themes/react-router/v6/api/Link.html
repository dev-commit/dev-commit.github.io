<Link />

Видоизмененная ссылка под React для редиректа на нужную страницу. Предотвращает обновление страницы, в отличии от обычной ссылки Местоположение ссылки Гарантирует, что этот компонент не будет сопоставлен как «активный», когда совпадают пути его потомков. Например, чтобы отобразить ссылку, которая активна только в корне веб-сайта, а не в каких-либо других URL-адресах

<Link />

import { Link } from 'react-router-dom'; const App = () = ( <> <Link to="/">Home</Link> <Link to="/about/">About</Link> </> );

Настройка сервера Webpack

  • При переходе по Link изменяется url, но физически этой страницы нет. Поэтому нужно настроить сервер
module.exports = { output: { filename: 'bundle.js', path: path.resolve(__dirname, 'build'), publicPath: '/' }, devServer: { contentBase: './dist', historyApiFallback: { disableDotRule: true }, stats: 'errors-only', port: 3000 } }
