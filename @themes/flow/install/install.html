Установка и настройка

1. Установка Flow

npm i --save-dev flow-bin

2. Добавление flow в секцию "scripts" файла package.json

{ // ... "scripts": { "flow": "flow", // ... }, // ... }

3. Запуск исходной конфигурации

npm run flow init

4. Установка пресета для работы с Flow

npm i --save-dev @babel/preset-flow

5. Добавление пресета в конфигурацию Babel, файл .babelrc

{ "presets": [ "@babel/preset-flow", "react" ] }
  • Для работы с Flow не требуется отдельно устанавливать пресет react - Flow уже понимает JSX-синтаксис. Тем не менее, часто используют оба пресета одновременно

6. Запуск Flow

npm run flow

7. Добавление аннотаций типов

  • По умолчанию Flow проверяет только файлы, содержащие специальную аннотацию (обычно её указывают в самом начале файла)
// @flow
