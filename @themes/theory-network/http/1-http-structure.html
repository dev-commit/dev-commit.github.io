Структура протокола

Каждое HTTP-сообщение состоит из трёх частей, которые передаются в указанном порядке:

  • 1. Starting Line (Стартовая строка) - определяет тип сообщения
  • 2. Headers (Заголовки) - характеризуют тело сообщения, параметры передачи и прочие сведения
  • 3. Message Body (Тело сообщения) - данные сообщения. Может отсутствовать

Структура Starting Line

Строка запроса
  • Method: HTTP Method GET, POST и т.д.
  • URI - определяет путь к запрашиваемому документу
  • HTTP/Version - н-р 1.0
GET /wiki/HTTP HTTP/1.0 Host: ru.wikipedia.org
Строка ответа
  • HTTP/Version - н-р 1.0 (как в запросе)
  • Status Code - HTTP Status Codes: 202, 404 и т.д.
  • Reason Phrase - текстовое пояснение к коду ответа для пользователя
HTTP/1.0 200 OK
