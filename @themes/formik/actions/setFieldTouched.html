Событие onBlur, setFieldTouched

  • Валидация происходит при onBlur
  • В стандартном состоянии валидация происходит при нажании на кнопку submit
const App = ({ setFieldTouched }) => { const handleBlur = (name) => { setFieldTouched(name, true); } return ( <Input onBlur={handleBlur('emailPersonal')} name='userName' /> ); };
