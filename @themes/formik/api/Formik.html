<Formik /> <Form /> <Field />

Пример

import { Formik, Form, Field } from "formik"; const App = () => { const initialValues = { userName: "" }; const handleSubmit = (values) => { console.log(values); // => { userName: "text" } }; return ( <Formik initialValues={initialValues} onSubmit={handleSubmit} > <Form> <Field type="text" name="userName" /> {/* Не сработает */} {/* <input type="text" name="userName" /> */} <button type="submit">Submit</button> </Form> </Formik> ); };

Примечание

  • Вместо <Field /> напрямую использовать <input /> не получится
  • Для <input /> нужно обработать событие onChange и записать это в Formik
  • Обрабтать можно, использовав дочерний компонент наподобие <Field /> в и нем изменить значение через хук useField()
<Field type="text" name="userName" /> {/* Не сработает */} <input type="text" name="userName" />
