Особенности Styled Components

  • Material UI завязан на Styled Components
  • Если использовать его с обычными CSS-modules, то стили, прокинутые в компоненты Material UI не будут применяться, т.к. они будут располагаться выше, чем стандартные стили Material UI
  • При использовании Styled Components, стили будут ниже и смогут переопределить стандартные стили
