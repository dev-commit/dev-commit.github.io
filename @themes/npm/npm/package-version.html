Установка версии пакета

npm i PackageName # npm i PackageName@4.16.1 # конкретная версия npm i PackageName@^ # последняя версия
  • ~ (тильда) подбирает последнюю минорную версию пакета (последнюю цифру), например ~7.3.3 найдет последнюю 7.3.x, допустим 7.3.12 но не 7.4.0
  • ^ (крышка) подбирает последнюю мажорную версию пакета (среднюю цифру), например ^7.3.3 найдет последнюю 7.x.x, допустим 7.4.0 но не 8.0.0
npm i PackageName
  • Пакета нет в package.json - установится последняя версия
  • Пакет есть в package.json PackageName": "1.0.0" - установится версия 1.0.0
  • Пакет есть в package.json PackageName": "^1.0.0" - установится последняя версия
