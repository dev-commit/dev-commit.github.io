Информация

Команды

  • Ctrl + Shift + P -> Package Control
  • Ctrl + Shift + P -> View: Toggle Menu - убрать / показать меню
  • Ctrl + Shift + D - продублировать строку
  • Ctrl + / - комментарий
  • Ctrl + KU - Uppercase
  • Ctrl + KL - Lowercase
  • Ctrl + F2 / Ctrl + Fn + F2 (для ноутбуков) - отметка (по ней же и снимается)

Package Control

  • Ctrl + Shift + P -> Install Package Control - установка Package Control
Preferences -> Package Control -> Install Package -> NamePackage Preferences -> Package Control -> Remove Package -> NamePackage
