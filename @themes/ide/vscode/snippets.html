Файл сниппета

{ "react": { "scope": "javascript,javascriptreact", "prefix": "component", "body": [ "import styles from './${1}.module.css';", "", "const ${1} = () => {", "\treturn (", "\t\t<>${2}", "\t)", "}", "", "export default ${1};", "" ] } }

Спецсимволы

  • \t - табуляция

Параметры

  • scope - ограничение для файлов (по умолчанию для всех)
  • prefix - вызов из файла
  • body - тело сниппета
  • description - описание сниппета
