Примечания

  • В пути проекта (терминала) не должно быть русскоязычных папок. Многие gulp пакеты не понимают русскоязычных путей
  • Запуск командной строки в директории проекта (shift+пкм)
  • После изменения gulpfile.js, нужно перезапускать сервер "gulp watch". В командной строке ctrl+c 2 раза - выйти из watch
  • Файлы из папки sass с нижним подчеркиванием "_part.scss" не участвуют в компиляции, их можно добавить только через import в другой sass файл @import "part", при этом можно не указывать нижнее подчеркивание и расширение
// При импорте css файла в sass не нужно писать расширение у css @import "app/libs/magnific-popup/dist/magnific-popup" gulp vinyl-ftp // загрузка на сервер
