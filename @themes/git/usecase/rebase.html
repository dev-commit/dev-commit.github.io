Rebase

  • Разрешение конфликтов разницы "develop" и "feature"
git fetch origin git rebase -i origin/develop
  • Если есть конфликты merge, откроется редактор
  • Пофиксить конфликты
  • Если несколько коммитов, то вызвать git rebase --continue и фиксим следующий коммит
  • Если есть конфликт, то его нужно фиксить в каждом коммите
git add . git rebase --continue git log git push --force git merge —abort git reset --hard HEAD git pull
