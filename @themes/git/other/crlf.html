"LF" и "CRLF"

При клонировании, Git меняет файлы с "LF" на "CRLF"

1. Проверить конфиг

git config core.autocrlf
  • Если false, то ок - иначе поменять
git config --global core.autocrlf false

2. В проекте есть файл .gitattributes

# Auto detect text files and perform EOL normalization * text=auto
  • Файл .gitattributes имеет более высокий приоритет, чем core.autocrlf
  • http://git-scm.com/docs/gitattributes
