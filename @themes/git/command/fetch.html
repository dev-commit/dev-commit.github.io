Получение актуальной информации о состоянии удаленных веток (указателях) на удаланном репозитории.
Обновление веток слежения remotes/origin
  • Fetch скачивает все ветки с удаленного репозитория.
  • С локальными указателями веток ничего не происходит.
  • Если кто-то сделал push в ветку - в этот момент fetch скачает недостающие коммиты и передвинет ветки слежения.
git fetch --all # скачать все ветки с "origin", но не мерджить их в локальный репозиторий git fetch origin # скачать ветку "master" с "origin", но не мерджить ее в локальный репозиторий git fetch origin master
