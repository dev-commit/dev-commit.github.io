Разница между рабочей директорией и индексом. Вычисление разницы между любыми двумя Git деревьями
git diff # показать изменения с момента последнего коммита git diff --staged # разница между индексом и последним коммитом (изменения, которые находятся в Индексе) git diff --cached # посмотреть, что добавлено в stage git diff master features # разница между любыми двумя коммитами "master" и "features" git difftool # запускает внешнюю утилиту сравнения для показа различий в двух деревьях
