Список веток с указанием текущей

Команды

Список веток

git branch # список веток с указанием текущей git branch -a # список веток с указанием remotes/origin

Создание новой ветки

git branch НазваниеВетки # создание новой ветки (ответвиться от текущей) git checkout -b НазваниеВетки # создание новой ветки и переключение на нее

Удаление ветки

# Удаление ветки из локального репозитория git branch --delete НазваниеВетки # удаление ветки git branch -d НазваниеВетки # удаление ветки (после мерджа) git branch -D НазваниеВетки # удаление ветки (тупиковая ветвь) # Удаление ветки из удаленного репозитория git push origin --delete НазваниеВетки git push origin :НазваниеВетки

Переименование локальной ветки

  • -m происходит от слова move (или mv)
  • mv используется для переименования файлов в Linux
git branch -m new-name # если находимся на ветке для переименования git branch -m old-name new-name # если находимся на другой ветке
