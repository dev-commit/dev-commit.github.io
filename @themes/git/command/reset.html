Отмена изменений. Изменяет указатель HEAD и, опционально, состояние индекса.
  • Сбросить коммиты до начального в ветке
  • Убрать из индексации все файлы и директории, которые возможно уже были проиндексировалы с помощью git add
git reset HEAD~ # отменить 1 коммит git reset HEAD~1 # отменить 1 коммит git reset HEAD^ # отменить 1 коммит git reset HEAD^1 # отменить 1 коммит git reset HEAD^2 # отменить 2 коммита git reset HashCommit # откатиться к конкретному коммиту (хэш из "git log") git reset HEAD ИмяФайла # если изменения были индексированны и нужно откатиться перед самим коммитом git reset --soft HEAD~ # откатиться к коммиту и ОСТАВИТЬ последующие git reset --hard HEAD~ # откатиться к коммиту и УДАЛИТЬ последующие
