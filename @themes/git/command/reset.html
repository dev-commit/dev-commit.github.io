Отмена изменений. Изменяет указатель HEAD и, опционально, состояние индекса
git reset git reset HEAD^2 # отменить 2 коммита git reset --hard HEAD^1 # откатиться на заданное кол-во коммитов git reset --hard HashCommit # откатиться к конкретному коммиту и удалить последующие (хэш смотрим в «git log») git reset HEAD ИмяФайла # если изменения были индексированны и нужно откатиться перед самим коммитом
