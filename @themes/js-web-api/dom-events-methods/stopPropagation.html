stopPropagation

Предотвращает дальнейшее распространение текущего события на фазах погружения и всплытия
  • Не предотвращает появление поведения по умолчанию; например, клики по ссылкам все еще обрабатываются (для этого нужен preventDefault())
  • Не предотвращает распространение на другие обработчики событий текущего элемента (для этого нужен stopImmediatePropagation)
