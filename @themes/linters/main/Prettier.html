Плагин для VSCode

  • Prettier - Code formatter

1. Установка плагина

  • Extensions ➝ Prettier - Code formatter ➝ Install ➝ Reload

2. Настройка плагина для определенных файлов

  • View ➝ Command Palette... ( Ctrl + Shift + P )
  • Ввести "Format Document"
  • Выбрать "Configure..."
  • Выбрать "Prettier - Code formatter"
  • В примере Prettier будет применяться ко всем файлам HTML во всех проектах

3. Применение Prettier при сохранение файла

  • File ➝ Preferences ➝ Settings
  • Text Editor ➝ Font ➝ Format On Save (checked)

4. Стандартный конфиг Prettier в VSCode

  • Если в проекте есть файл .prettierrc.json, то конфиги будут браться из него, а не из настроек VSCode
  • File ➝ Preferences ➝ Settings
  • Extensions ➝ Prettier

Файл .prettierrc.json

  • Если в проекте есть файл .prettierrc.json, то конфиги будут браться из него
{ "trailingComma": "es5", "tabWidth": 4, "semi": false, "singleQuote": true }
  • Если используется npm, то можно устновить пакет с конфигами и подключить его в package.json
{ "devDependencies": { "prettier-config-nick": "^1.0.2", }, "prettier": "prettier-config-nick" }
  • Если в проекте нет файла .prettierrc.json, то будут браться стандартные конфиги из IDE

Файл .prettierignore

  • src - запретить директорию "src"
  • *.html - запретить файлы html
  • * - запретить все файлы
