Prettier - Code formatter

https://prettier.io/

Плагин для VSCode

  • Prettier - Code formatter

1. Установка плагина

  • Extensions ➝ Prettier - Code formatter ➝ Install ➝ Reload

2. Настройка плагина для определенных файлов

  • View ➝ Command Palette... ( Ctrl + Shift + P )
  • Ввести "Format Document"
  • Выбрать "Configure..."
  • Выбрать "Prettier - Code formatter"
  • В примере Prettier будет применяться ко всем файлам HTML в проекте для которого была запущена Command Palette

3. Применение Prettier при сохранение файла

  • File ➝ Preferences ➝ Settings
  • Text Editor ➝ Font ➝ Format On Save (checked)

Файл .prettierignore

  • src - запретить директорию "src"
  • *.html - запретить файлы html
  • * - запретить все файлы
