Циклы

Цикл for

for (let i = 1; i < 10; i++) { console.log(i); }

Цикл while

let i = 0; while (i < 10) { console.log(i); i++; }

Цикл do while

let i = 0; do { console.log(i); i++; } while (i < 10); break // выход из цикла continue // прекращает выполнение текущей итерации цикла и переходит на следующую
