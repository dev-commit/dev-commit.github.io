Информация

  • Массив - это объект, где в качестве ключей выбраны цифры, с дополнительными методами и свойством length
  • Так как это объект, то в функцию он передаётся по ссылке
  • Нет причин использовать массив как обычный объект, хотя технически это и возможно

Объявление

const arr = []; // Пустой массив const arr = ['Tony', 'Anne']; // Массив из заполненных элементов const arr = [,,'Tony']; // Пропуск элементов в массиве

Конструктор

const arr = new Array(5); // Пустой массив в длиной 5 const arr = new Array('Tony', 'Anne'); // Массив из заполненных элементов

Массив обьектов

const a = [ { name: 'Tony', age: 23 }, { name: 'Jack', age: 25 } ]; console.log(a[0].name); // 'Tony' console.log(a[1].age); // 25
