includes()

  • Метод includes() определяет, содержит ли массив определённый элемент, возвращая в зависимости от этого true или false
const arr = [1, 2, 3]; console.log(arr.includes(2)); // => true console.log(arr.includes(5)); // => false const arr = ['cat', 'dog', 'bat']; console.log(arr.includes('cat')); // => true console.log(arr.includes('at')); // => false
