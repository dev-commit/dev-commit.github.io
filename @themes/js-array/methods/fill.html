Array.prototype.fill()

Метод заполняет все элементы массива от начального до конечного индексов одним значением. Значение, заполняющее массив Необязательный параметр. Начальный индекс Необязательный параметр. Конечный индекс Изменённый массив
[1, 2, 3].fill(4); // [4, 4, 4] [1, 2, 3].fill(4, 1); // [1, 4, 4] [1, 2, 3].fill(4, 1, 2); // [1, 4, 3]
