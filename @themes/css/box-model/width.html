width

fit-content

  • Ширина блока по ширине контента
div { width: fit-content; }

width min

  • Использование width min вместо width и max-width
.item { /* width: 50%; */ /* max-width: 20rem; */ width: min(50%, 20rem); }

width auto

  • Возвращение ширины к исходному значению
div { width: auto; }
