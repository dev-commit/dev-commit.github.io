position

  • static - статичное (стандартное позиционирование)
  • relative - относительное (положение устанавливается относительно его исходного места)
  • absolute - абсолютное (отсчёт координат ведётся от края окна браузера, если у родителя не установлено position: relative)
  • fixed - фиксированное (привязывается к указанной свойствами left, top, right и bottom точке на экране)
  • sticky - сочетание относительного и фиксированного позиционирования
div { position: static; position: relative; position: absolute; position: fixed; position: sticky; }

Особенности

  • static и relative сохраняют своё естественное положение в потоке документа, в то время как absolute и fixed «вырываются» из потока документа и становятся плавающими

sticky

  • Элемент с position sticky может перемещаться только в пределах контейнера, в котором находится
<!-- НЕ РАБОТАЕТ!!! --> <style> .sticky { position: sticky; top: 0; } </style> <div class="wrapper"> <div class="sticky"> Контент </div> </div>
  • «Липкое» позиционирование состоит из двух основных частей: «липкого» элемента и «липкого» контейнера
  • «Липкий» элемент — это элемент, которому задали position: sticky. Элемент будет становиться плавающим, как только область видимости достигнет определённой позиции, например top: 0px
  • «Липкий» контейнер — это HTML-элемент, который оборачивает «липкий» элемент. Это максимальная область, в которой может перемещаться наш элемент
  • При задании элементу position: sticky, его родитель автоматически становится «липким» контейнером!
  • Контейнер будет являться областью видимости для элемента. «Липкий» элемент не может выйти за пределы своего «липкого» контейнера

Задачи

Определение вычисленного значения

  • Вопрос: Какое вычисленное значение для свойства display будет в коде
span { display: inline-block; position: absolute; }
  • Ответ: block. При использовании position: absolute браузер делает этот элемент блочным
