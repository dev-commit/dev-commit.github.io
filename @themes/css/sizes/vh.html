Viewport width (vw). Viewport height (vh)

div { width: 100vw; - ширина блока 100% от viewport height: 100vh; - высота блока 100% от viewport } vh, vw - единицы измерения, которые расчитываются относительно viewport - включают в себя ширину скроллбара vmin, vmax div { width: 100vmin; height: 100vmin; }
