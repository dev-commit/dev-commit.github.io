CSS Blend Modes

• normal # обычный • multiply # умножение • screen # осветление • overlay # перекрытие • darken # замена тёмным • lighten # замена светлым • color-dodge # осветление основы • color-burn # затемнение основы • hard-light # направленный свет • soft-light # рассеянный свет • difference # разница • exclusion # исключение • hue # тон • saturation # насыщенность • color # цвет • luminosity # яркость

background-blend-mode

background-blend-mode [codepen] background-blend-mode playground
  • Управляет режимами наложения слоев фона, заданного в CSS
  • Слои: backround-color, background-image (несколько), linear-gradient
div { background-color: blue; background-image: url(...); background-blend-mode: multiply; }
