Font / Text

Свойства

font-*

p { font-size: 20px; # размер шрифта font-style: italic; # начертание шрифта font-weight: bold; # насыщенность шрифта }

text-*

p { text-align: center | left | right; # горизонтальное выравнивание текста в пределах элемента text-indent: 20px; # красная строка text-transform: uppercase | lowercase; # преобразованием текста элемента в заглавные или прописные символы }

spacing

div { word-spacing: 10px; # расстояние между словами letter-spacing: 4px; # расстояние между буквами line-height: 1.5; # расстояние между строками }

CSS Initial Letter

p {     initial-letter: size position; } # size - сколько строк занимает символ # position - с какой строки он начинается p::first-letter {     initial-letter: 4;     initial-letter: 4 1; }

Примеры

Обрезать текст по длине блока

div { white-space: nowrap; # запрещаем перенос строк white-space: pre-wrap; # разрешить перенос строк в pre overflow: hidden; # обрезаем все, что не помещается в область text-overflow: ellipsis; # добавляем многоточие }

Шрифты

Variable Fonts

  • Один шрифт, множество начертаний

Виды

  • Sans-serif - без засечек
  • Serif - с засечками
  • Accident - декоративный
  • Handwriting - рукописный
  • Monospaced - моноширинный
  • Symbol - символьный: fontawesome, stroke, icomoon

Безопасные шрифты

  1. Open Sans
  2. Museo Sans
  3. Roboto
  4. Verdana
  5. AG Book

Подключние

  • Расширения: .eot, .ttf, .woff
@font-face { font-family: OpenSans; - имя шрифта src: url(../font/OpenSans.ttf); - путь к файлу со шрифтом }
