RGB

rgb( red, # 0 - 255 или 0% - 100% green, # 0 - 255 или 0% - 100% blue, # 0 - 255 или 0% - 100% alpha # 0 - 1 );
  • red (красный) - интенсивность цвета
  • green (зеленый) - интенсивность цвета
  • blue (синий) - интенсивность цвета
  • alpha (прозрачность) - альфа-канал

RGB цвета

div { color: rgb(255, 0, 0); } /* red */ div { color: rgb(0, 255, 0); } /* green */ div { color: rgb(0, 0, 255); } /* blue */

RGBA цвета

  • Значения цвета RGBA являются расширением значений цвета RGB с альфа-каналом, который определяет непрозрачность объекта
  • RGBA цвет задается с помощью функции RGBA(), которая имеет синтаксис: rgba(red, green, blue, alpha)
  • Параметр Alpha является числом между 0,0 (полностью прозрачным) и 1,0 (полностью непрозрачным)
rgba(red, green, blue, alpha); div { color: rgba(255, 0, 0, 0.3); } /* red with opacity */ div { color: rgba(0, 255, 0, 0.3); } /* green with opacity */ div { color: rgba(0, 0, 255, 0.3); } /* blue with opacity */
