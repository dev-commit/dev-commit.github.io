Значения для цветов

currentColor

  • Значение background-color будет меняться при изменени значения color
  • Свойство currentColor автоматически подставляет значение свойства color для текущего элемента
.link { color: blue; } .link::before { content: 'before'; background-color: currentColor; }

transparent

  • Цвет по умолчанию
div { color: transparent; background-color: transparent; border: 2px solid transparent; }
