HSL

hsla( hue, # 0 - 360 градусов saturation, # 0% - 100% lightness, # 0% - 100% alpha # 0 - 1 );
  • hue (оттенок) - позиция цвета на цветовом круге. 0 (или 360)-красный, 120-зеленый, 240-синий
  • saturation (насыщенность) - насколько насыщенным (ярким) будет цвет. 0% означает оттенок серого и 100%-полный цвет
  • lightness (осветление) - насколько светлым или темным будет цвет. 0%-черный, 100%-белый
  • alpha (прозрачность) - альфа-канал

HSL цвета

div { color: hsl(120, 100%, 50%); } /* green */ div { color: hsl(120, 100%, 75%); } /* light green */ div { color: hsl(120, 100%, 25%); } /* dark green */ div { color: hsl(120, 60%, 70%); } /* pastel green */

HSLA цвета

  • Значения цвета HSLA являются расширением значений цвета HSL с альфа-каналом, который определяет непрозрачность объекта
  • Параметр Alpha является числом между 0,0 (полностью прозрачным) и 1,0 (полностью непрозрачным)
hsla(hue, saturation, lightness, alpha); div { color: hsla(120, 100%, 50%, 0.3); } /* green */ div { color: hsla(120, 100%, 75%, 0.3); } /* light green */ div { color: hsla(120, 100%, 25%, 0.3); } /* dark green */ div { color: hsla(120, 60%, 70%, 0.3); } /* pastel green */
