text-decoration-thickness

  • Задает толщину линии для text-decoration отличного от "none"
a { text-decoration-thickness: 3px; }

text-underline-offset

  • Задает отступ от текста до underline линии для text-decoration отличного от "none"
a { text-decoration: underline; text-underline-offset: 5px; }
