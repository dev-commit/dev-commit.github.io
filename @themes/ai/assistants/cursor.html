Информация

  • Fork VSCode

Элементы

Tab

  • Автодополнение кода, которое предлагает многострочные изменения. Нажмите Tab, чтобы принять предложения на основе текущего кода и недавних правок.
  • Можно отключить

Agent

  • ИИ, который читает и изменяет код в нескольких файлах. Опишите изменения простым языком — Agent выполнит их.
  • Agent — это ассистент Cursor, который может самостоятельно выполнять сложные задачи разработки, запускать команды в терминале и редактировать код. Откройте в боковой панели с помощью

Inline Edit

  • Редактирование выделенного кода на естественном языке
  • Позволяет описать изменения и применить их прямо на месте

Чат

  • Интерфейс для взаимодействия с ИИ
  • Поддерживает несколько вкладок, историю чатов, контрольные точки и экспорт
