Операции

https://cursor.com/ru/docs/get-started/concepts
  • Tab - автодополнение кода, которое предлагает многострочные изменения. Нажмите Tab, чтобы принять предложения на основе текущего кода и недавних правок.
  • Agent - ИИ, который читает и изменяет код в нескольких файлах. Опишите изменения простым языком — Agent выполнит их.
  • Inline Edit - редактирование выделенного кода на естественном языке. Нажмите Cmd+K, чтобы описать изменения и применить их прямо на месте.
  • Чат - интерфейс для взаимодействия с ИИ. Поддерживает несколько вкладок, историю чатов, контрольные точки и экспорт.
