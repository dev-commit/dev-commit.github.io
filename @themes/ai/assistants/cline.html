Add Context

  • @url - вставьте URL-адрес расширения для загрузки и преобразования в Markdown. Это полезно, когда вы хотите предоставить Cline последние документы.
  • @problems - добавляет ошибки и предупреждения в рабочей области (панель "Problems"), которые Cline может исправить.
  • @file - добавляет содержимое файла, чтобы вам не приходилось тратить время на запросы API для подтверждения чтения файла (+ ввод для поиска файлов).
  • @folder - добавляет все файлы из папки одновременно для ускорения работы.
  • @terminal - ссылка на содержимое Терминала
  • @git - ссылка на “Working Changes” и “Commits”

Файл .clinerules

  • Добавьте файл .clinerules в корневой уровень, чтобы указать пользовательские инструкции для проекта.

Переключение режима “Plan Mode” и “Act Mode”

1. Plan Mode (Режим планирования)
  • В этом режиме я помогаю спланировать решение задачи
  • Мы можем обсудить детали, требования и подход к реализации
  • Я могу задавать уточняющие вопросы и предлагать различные варианты решения
  • Этот режим полезен для сложных задач, требующих предварительного планирования
2. Act Mode (Режим действия)
  • В этом режиме я непосредственно выполняю задачи
  • Могу писать код, создавать файлы, запускать команды
  • Работаю пошагово, подтверждая успешность каждого действия
  • Этот режим используется для реализации уже спланированного решения

Диаграммы Mermaid

  • Диаграммы Mermaid в режиме "Plan mode"
