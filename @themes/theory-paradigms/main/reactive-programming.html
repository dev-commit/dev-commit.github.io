Реактивное программирование

  • Reactive Programming - набор подходов пытающихся решить проблему Side Effects
  • Самой известной реализацией этой парадигмы является библиотека RxJS, использующая концепцию Reactive Streams
  • Excel - как реактивная среда разработки
  • При задании формул, значения в ячейках зависят друг от друга. Это реактивное программирование в действии. При изменении значения в одной ячейке, зависимые ячейки пересчитываются автоматически
  • React построен на парадигме реактивного программирования
  • Этот декларативный подход предлагает описывать данные в виде набора утверждений
  • Изменение одного из параметров ведёт за собой автоматический пересчёт всех зависимостей
Пример
a = b + c
  1. В императивном программировании выражение означает: взять текущие значения b и c, сложить их и присвоить a результат. Дальнейшие изменения параметров b и c не влияют на a
  2. В реактивном программировании выражение означает: a есть сумма b и с. Если b или c изменятся в будущем, это отразится на величине a
