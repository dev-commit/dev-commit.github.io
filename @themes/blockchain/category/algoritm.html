Вариант 1 - Frontend Only

Frontend напрямую обращается к Blockchain со Smart Contracts как к API и получает данные:

1 Приложение через библиотеку Web3.js делает запрос к Wallet (кошельку) для авторизации пользователя. В кошельке хранится Etherium-аккаунт (например) пользователя для Blockchain

2 После авторизации, приложение может обращаться к Blockchain Server. Для обращения к Blockchain должна быть своя Node Blockchain - свой компьютер, подключенный к Blockchain сети и имеющей копию Etherium (например).

Можно не создавать свою Node Blockchain, а воспользоваться API для доступа к Blockchain и облачным протоколам хранения:

  • Infura
  • Moralis
  • Alchemy
