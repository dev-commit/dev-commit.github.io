Два варианта работы с Blockchain

  • 1. Frontend Only. Frontend напрямую обращается к Blockchain со Smart Contracts как к API и получает данные
  • 2. Backend Only. Frontend работает с API сервера, не взаимодействуя с Blockchain напрямую. Application Server (Backend) взаимодействует с Blockchain и БД
