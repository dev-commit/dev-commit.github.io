Metamask

https://metamask.io/
  • Metamask - отдельная программа / плагин для браузеров / крипто-кошелек
  • Позволяет авторизовать Web 3.0 приложение в Blockchain сети при помощи аккаунта
  • В MetaMask можно создать кошелек (аккаунт) Ethereum и использовать этот аккаунт для авторизации
  • Позволяет подключить Сhrome-браузер к сети Blockchain
