Ренденинг

  1. requestAnimationFrame - соответствует обновлением экрана экрана. Анимация элементов DOM. Передаем callback, который нужно вызвать перед рендером
  2. Style Calculation - вычисление стилей для каждого элемента
  3. Layout Calculation - вычисление взаимного расположения и размеров элементов на странице, плюс компоновка всего
  4. Painting - превращение всего этого в пиксельную картинку
