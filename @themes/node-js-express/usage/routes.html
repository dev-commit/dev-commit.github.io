Маршрутизация

Структура

1. Подключение и вызов Express

const express = require('express'); const app = express();

2. Марштуризация

app.get('/url', (req, res) => { // Отображаение текста res.send('This is Text'); // Отображаение HTML res.send('<h1>This is HTML</h1>'); // Отображаение HTML файла res.sendFile(__dirname + '/index.html'); });

3. Прослушивание порта

const PORT = process.env.PORT || 3000; app.listen(PORT, '127.0.0.1');

Маршрутизация

1. Статическая ссылка

// главная app.get('/', (req, res) => { res.send('This is Index'); }); // news app.get('/news', (req, res) => { res.send('This is News'); });

2. Динамические параметры в URL

// > "/news/starter" app.get('/news/:name', (req, res) => { const params = req.params; // => { name: 'starter' } }); // > "/news/starter/23" app.get('/news/:name/:id', (req, res) => { const params = req.params; // => { name: 'starter', id: '23' } });

3. Query параметры из URL

// > "/news?id=23&city=Melbourne" app.get('/news', (req, res) => { const query = req.query; // => { id: '23', city: 'Melbourne' } });
