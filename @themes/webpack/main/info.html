Информация

  • Webpack - модульный сборщик, который позволяет объединять ресурсы и библиотеки, необходимые для проекта, в один файл. В частности, он анализирует JS-код (а также CSS или другой, через лоадеры), это даёт ему superpowers
  • Gulp/Grunt - общая система для организации задач, которая сама по себе сборку не поддерживает, но её можно на ней написать, с помощью плагинов. При этом анализа кода, скорее всего, не будет, а значит ряд продвинутых фич отпадут
  • Сборка («бандлинг») - это процесс выявления импортированных файлов и объединения их в один «собранный» файл (часто называемый «bundle» или «бандл»). Этот бандл после подключения на веб-страницу загружает всё приложение за один раз
