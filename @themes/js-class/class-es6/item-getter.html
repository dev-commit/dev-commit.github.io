Геттеры и Сеттеры

  • Геттеры и Сеттеры - аксессоры. Внутри класса выглядят и ведут себя как методы, а снаружи выглядят и ведут себя как свойства
  • Эти свойства-методы позволяют получить доступ и присвоить значения "настоящим" свойствам объекта, т.е. выступают в роли фильтра
  • Названия свойств get и set не должны совпадать с названиями основных свойств объекта
  • Свойство get связывает свойство объекта с функцией, которая будет вызываться при обращении к этому свойству
class Task { constructor() { this._done = false; } get done() { return this._done === true ? 'Yes' : 'No'; } set done(value) { if(value != undefined && typeof value === 'boolean') { this._done = value; } else { console.error('Ошибка! Укажите значение true или false'); } } complete() { this.done = true; } } const task = new Task(); // Обращение через Геттер const a1 = task.done; // => "No" // Обращение напрямую к свойству const a2 = task._done; // => false // Установка значения через Сеттер task.done = true; // Обращение через Геттер const a3 = task.done; // => "Yes"
