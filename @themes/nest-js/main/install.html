Установка и настройка

1. Установить Node.js

https://nodejs.org/en/

2. Установить глобально NestJS

npm i --g @nestjs/cli

3. Создать проект

  • Запускать консоль под Администратором
  • Будет предложен выбор пакетного менеджера (npm, yarn, .etc)
nest new НазваниеПроекта

4. Запустить сервер в режиме dev

  • http://127.0.0.1:3000/
npm run start:dev
