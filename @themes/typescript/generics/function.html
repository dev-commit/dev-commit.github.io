Дженерик и стрелочная функция

.TS файл

const foo2 = <T>(props: T): void => { // }; function foo1<T>(props: T): void { // }

.TSX файл

  • В файлах .TSX есть особенность парсинга. Если указать Дженерик для стрелочной функции, то будет ошибка
// Не сработает в TSX файле const foo = <T>(data: T) => { // } foo<SomeInterface>(data);

Решение

1. Extends trick

const foo = <T extends unknown>(data: T) => { // } const foo = <T extends any>(data: T) => { // }

2. Comma trick

const foo = <T,>(data: T) => { // }

3. Just use a regular function

function foo<T>(data: T) { // }
