Базовые

Типы данных

  • boolean - логическое значение true или false
  • number - числовое значение
  • string - строки
  • array - массивы
  • tuple - кортежи
  • enum - перечисления
  • any - произвольный тип
  • null и undefined - соответствуют значениям null и undefined в JavaScript
  • void - отсутствие конкретного значения, используется в основном в качестве возвращаемого типа функций
  • never - представляет отсутствие значения и используется в качестве возвращаемого типа функций, которые генерируют или возвращают ошибку

Примеры

const str: string = 'Hello' const bool: boolean = true const int: number = 42 const float: number = 4.2 const num: number = 3e10 const nubmerArray: number[] = [1, 2, 3] const nubmerArray: Array<number> = [1, 2, 3] // Дженерик типы - указывает из чего состоит данный объект const nubmerArray: string[] = ['Hello', 'World']
