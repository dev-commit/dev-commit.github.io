Базовые

String

const str: string = 'Hello'

Boolean

const bool: boolean = true

Number

  • Все числа в TypeScript, как и в JavaScript, являются числами с плавающей точкой

Поддержка систем счисления

  • Двоичная
  • Восьмеричная
  • Десятичная
  • Шестнадцатиричная
const decimal: number = 42; const decimal: number = 4.2; const hex: number = 0xf00d; const binary: number = 0b1010; const octal: number = 0o744;

Array

  • Массивы являются строго типизированными: если изначально массив содержит строки, то в будущем он сможет работать только со строками
const list: number[] = [10, 20, 30]; const colors: string[] = ['red', 'green', 'blue']; console.log(list[0]); console.log(colors[1]);

Применение типа Array<> (Дженерик)

  • Дженерик указывает из чего состоит данный объект
const names: Array<string> = ['Tom', 'Bob', 'Alice'];

Object

  • На второй строке будет ошибка, поскольку компилятор после первой строки предполагает, что объект person будет иметь два свойства name и age
  • Должно быть соответствие по названиям, количеству и типу свойств
let person = { name: 'Tom', age: 23 }; person = { name: 'Alice' };

null и undefined

let a: undefined = undefined; let b: null = null;
  • Фактически можно присваивать значения undefined и null переменным других типов, например, number
  • В этом плане null и undefined выступают как подтипы других типов и полезны преимущественно в каких-то операциях, где неизвестен результат - то ли это будет число или строка, то ли это будет null. В этом случае, чтобы избежать возможной ошибки, мы можем проверить значение на undefined или null, как в JavaScript
let x: number = undefined; console.log(x); x = null; console.log(x); x = 5; console.log(x);
