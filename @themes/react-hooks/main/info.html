Информация

Информация

  • Хуки (React 16.8) - функции JavaScript, которые позволяют использовать состояние, методы жизненного цикла и другие возможности React из функциональных компонентов
  • Хуки позволяют вам повторно использовать логику состояния, не затрагивая дерево компонентов. Благодаря этому, хуки легко использовать в разных компонентах и делиться ими с сообществом
  • Хуки работают рядом с существующим кодом, поэтому вы можете внедрять их постепенно
  • Не работают внутри классов

Правила хуков

Можно использовать внутри:

  • Функциональные компоненты
  • Пользовательские хуки

Нельзя использовать внутри:

  • Классы
  • Обычные JavaScript-функции
  • Циклы
  • Условия
  • Вложенные функции

Преимущества

  • Делают переиспользование удобнее
  • Позволяют писать более интуитивно-понятный код
  • Избавляют от жизненного цикла

Решаемые проблемы

  • Трудно повторно использовать логику состояний между компонентами. В React нет способа «присоединить» повторно используемое поведение к компоненту (например, подключение к хранилищу)
  • Сложные компоненты становятся трудными для понимания. Каждый метод жизненного цикла часто содержит смесь несвязанной логики. Хуки позволяют разбить один компонент на маленькие функции по их назначению (например, подписке или загрузке данных), а не на основе методов жизненного цикла. Вы также можете контролировать внутреннее состояние с помощью редюсера, чтобы поведение было более предсказуемым
  • Классы создают существенный барьер в изучении React. Хуки позволяют использовать больше возможностей React без написания классов. Концептуально, React-компоненты всегда были ближе к функциям
