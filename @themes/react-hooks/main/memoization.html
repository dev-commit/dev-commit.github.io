useCallback vs useMemo

  • Когда мы рендерим компонент, то постоянно создается новая ссылка
  • useCallback - мемоизирует функцию: сохраняет ссылку на нее
  • А если ссылку мемоизировать и прокинуть в дочерний компонент, обернутый в React.memo, то у дочернего компонента не будет перерендера
  • Если не использовать useCallback, то даже с React.memo будет происходить перерендер дочернего компонента, т.к. каждый раз будет приходить новая ссылка
  • useCallback полезен, когда переписовка компонента занимает большое количество времени
