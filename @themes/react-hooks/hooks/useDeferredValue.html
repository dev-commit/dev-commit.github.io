Хук useDeferredValue()

Позволяет отложить обновление части UI значение, которое вы хотите отложить. Может иметь любой тип Во время первоначального рендеринга возвращаемое отложенное значение будет таким же, как и предоставленное вами значение. Во время обновлений React сначала попытается выполнить повторный рендеринг со старым значением (поэтому он вернет старое значение), а затем попытается выполнить еще один повторный рендеринг в фоновом режиме с новым значением (поэтому он вернет обновленное значение)
import { useState, useDeferredValue } from 'react'; const App = () => { const [query, setQuery] = useState(''); const deferredQuery = useDeferredValue(query); // ... }
