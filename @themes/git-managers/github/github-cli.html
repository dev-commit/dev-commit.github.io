GitHub CLI

Команды

Общее

gh auth login # Аутентификация gh repo clone UserName/RepoName # Клонирование репозитория gh commands # Доступные команды code . # Открыват проект в редакторе по умолчанию

Issue

gh issue status # Issue статус gh issue list # Список issue

Pull Request

gh pr create # Создает PR gh pr diff # Показывает изменения в PR gh pr review # Добавить апрув / коммент на ревью gh pr merge # Merge PR
