Виды веб приложений

  • Стандартные HTML страницы
  • SPA - Single-Page Application (одностраничное приложение)
  • PWA - Progressive Web App (веб-приложение для мобильных устройств)
  • SSR - Server-Side Rendering (рендеринг на стороне сервера с использованием Node.js на сервере)
  • SSG - Static Site Generation (генерация страниц без использования Node.js на сервере)
