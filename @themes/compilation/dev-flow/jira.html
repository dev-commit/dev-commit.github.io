Жизненный цикл (workflow) задачи

› OPEN # PM создал задачу › BACKLOG # Задача без оценки › TODO # Задача назначена на разработчика › IN PROGRESS # Начало работы над задачей › BLOCKED # Блокер для задачи › REVIEW # Завершена, на ревью › READY FOR QA # После ревью и merge в develop › IN QA / TESTING # На тестировании › REJECTED # От тестировщика в случае нахождения багов › IN DEV # На DEV-стенде › READY FOR STAGE # Готова к заливке на stage-стенд › ON STAGE # На STAGE-стенде › PENDING RELEASE # Готова к релизу › DONE / CLOSED / RESOLVED # Закрыта › REOPENED # PM заново открыл задачу
