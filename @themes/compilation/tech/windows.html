Задачи

1. Как сделать, чтобы ноутбук работал и не выключался при закрытии крышки в Windows 10

  1. Win + R
  2. powercfg.cpl
  3. В открывшемся окне настроек электропитания, слева, нажмите по пункту «Действие при закрытии крышки»

2. Добавление ярлыка для файла на панель задач

  1. Создать ярлык файла
  2. Открыть свойства ярлыка
  3. В поле объект прописать explorer: explorer C:\Users\anduser\Desktop\1.bat

Установка Windows

  • Невозможно установить Windows в раздел 1 диска 0
  • Установка Windows на данный диск невозможна.
  • На выбранном диске находится таблица MBR-разделов.
  • В системах EFI Windows можно установить только на GPT-диск.
