React.cloneElement

  • Реакт элементы нельзя изменять после того как они были созданы, но можно создавать модифицированные копии при помощи React.cloneElement()
Клонирует и возвращает новый React элемент, используя элемент в качестве отправной точки. Полученный элемент будет иметь пропсы исходного элемента, а новые пропсы будут поверхностно слиты воедино. Новые дочерние элементы заменят существующие. key и ref из исходного элемента будут сохранены элемент для клонирование props children Новый React-элемент
