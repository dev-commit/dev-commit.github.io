Установка пароля на Zip-архив

zip -ejr [путь/имя_архива] [путь_к_файлу_или_папке] zip -ejr ~/Desktop/test.zip ~/Desktop/test С ключом -ejr были ошибки при добавлении вложенных директорий. Поэтому оставил ключ -er zip -er ~/Desktop/test.zip ~/Desktop/test

Ключ "-ejr"

  • е - выполнить шифрование содержимого архива паролем
  • j - отбросить путь к корневой директории, т.е. архивировать только её содержимое без учета самой директории
  • r - рекурсивное добавление содержимого директории
