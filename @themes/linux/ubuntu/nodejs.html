Node.js

Установка

sudo apt-get install nodejs sudo apt-get install npm

Установка последней версии

# https://nodejs.org/dist/latest/ # из папки с архивом sudo tar -C /usr/local --strip-components 1 -xzf node-v12.2.0-linux-x64.tar.gz

Удаление

sudo apt-get remove nodejs sudo apt-get remove npm Полностью удалить nodejs, npm и node в ubuntu 14.04
