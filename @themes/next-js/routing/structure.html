Структура

  • Роутинг создается через файловую систему

Структура файлов

+ pages # Стандартная папка для роутов --------------------------------------------------------------------------- + about # Группировка роутов по категории "about" - author.js # Роут http://.../about/author - index.js # Роут http://.../about --------------------------------------------------------------------------- + post # Группировка роутов по категории "post" - [id].js # Динамический роут: http://.../post/42 --------------------------------------------------------------------------- - index.js # Точка входа (корневой роут) --------------------------------------------------------------------------- - posts.js # Роут: http://.../posts

Структура стандартного компонента-роута

  • React не испортируется
  • Название компонента не имеет значения. Главное, что export default
const App = () => { return (<h1>Hello</h1>) } export default App;
