Custom `App`

https://nextjs.org/docs/advanced-features/custom-app
  • Next.js использует компонент App для инициализации страниц
  • Файл: ./pages/_app.js
    Можно контролировать:
  • Сохранение макета между изменениями страницы
  • Сохранение состояния при навигации по страницам
  • Пользовательская обработка ошибок с использованием componentDidCatch
  • Вставить дополнительные данные на страницы
  • Добавить глобальный CSS
./pages/_app.js
  • В примере добавляются глобальные стили для всего приложения
  • Component - отображаемая страница
  • pageProps - объект с исходными реквизитами, которые были предварительно загружены для страницы
import '../styles/main.css'; const MyApp = (props) => { const { Component, pageProps } = props; return ( <> <Component {...pageProps} /> </> ) } export default MyApp; body { font-family: 'Bakbak One', cursive; }
