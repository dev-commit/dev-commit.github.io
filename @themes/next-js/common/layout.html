Создание Layout

  • Обычный React-компонент, в который оборачивается страница или ее часть
  • Обычно находится в корне в директории components
  • Подключается по необходимост на любую страницу

Объявление Layout

components/MainLayout.js
  • Для примера, добавлен Head и Link, но может быть что угодно
import Head from 'next/head'; import Link from 'next/link'; const MainLayout = (props) => { const { children, title = 'Next App' } = props; return ( <> <Head> <title>{title}</title> </Head> <nav> <Link href="/">Home</Link><br/> </nav> <main> {children} </main> </> ) } export default MainLayout;

Использование Layout

import MainLayout from '../components/MainLayout'; const App = () => { return ( <MainLayout title='Helo'> <h1>App</h1> </MainLayout> ) } export default App;
