Информация

  • В компоненте можно использовать стандартные Fetch для запросов на сервер. Но, в этом случае, запросы будут отправляться со стороны клиента в runtime. Если посмотреть исходный код страницы, то данных из API не будет.
  • Задача в том, чтобы поисковые роботы могли индексировать данные с сервера.
  • Для этого на сервере необходимо сделать запрос и вернуть уже страницу с добавленными данными.
  • Next.js вместо браузерного Fetch, использует пакет isomorphic-unfetch https://github.com/developit/unfetch
