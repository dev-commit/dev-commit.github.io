  • %c%s - позволяет стилизовать элементы

Один элемент

console.log( '%c%s', 'background: black; color: white; border-radius: 5px; padding: 5px; font-weight: bold', 'Hello', );

Два элемента

console.log( '%c%s%c%s', 'background: black; color: white; border-radius: 5px; padding: 5px; font-weight: bold', 'Hello1', 'background: pink; border-radius: 5px; padding: 5px; font-weight: bold', 'Hello2', );
