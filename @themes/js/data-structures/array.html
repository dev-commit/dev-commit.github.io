Массивы

  • Массив - это объект, где в качестве ключей выбраны цифры, с дополнительными методами и свойством length
  • Так как это объект, то в функцию он передаётся по ссылке
  • Нет причин использовать массив как обычный объект, хотя технически это и возможно

Объявление

const arr = []; // объявление пустого массива const arr = ['Tony', 'Anne']; // объявление массива из заполненных элементов const arr = [,,'Tony']; // пропуск элементов const arr = new Array(5); // объявление массива из пустых элементов с длиной 5 (конструктор) const arr = new Array('Tony', 'Anne'); // объявление массива (конструктор)

Методы

  • Получить плоский массив из объекта

Array.prototype.flatMap()

const checkedRows = { '0': [1,2,3], '1': [4,5,6], } const b = Object.values(checkedRows).flatMap(value => value); // => [1, 2, 3, 4, 5, 6]

push pop unshift shift

arr.push('Jack', 'Tony'); // добавляет элементы в конец arr.unshift('Jack', 'Tony'); // добавляет элементы в начало arr.pop(); // удаляет и возвращает последний элемент arr.shift(); // удаляет и возвращает первый элемент

includes

  • Метод includes() определяет, содержит ли массив определённый элемент, возвращая в зависимости от этого true или false
const arrNumbers = [1, 2, 3]; console.log(arrNumbers.includes(2)); // => true console.log(arrNumbers.includes(5)); // => false const arrPets = ['cat', 'dog', 'bat']; console.log(pets.includes('cat')); // => true console.log(pets.includes('at')); // => false

indexOf lastIndexOf

const a = colors.indexOf('white'); // возвращает первый индекс, по которому элемент может быть найден в массиве или -1, если индекса нет const a = colors.lastIndexOf('white'); // возвращает последний индекс, по которому данный элемент может быть найден в массиве или -1, если такого индекса нет

join split

const str = arr.join('.'); // преобразование массива в строку с заданным разделителем const arr = str.split(';'); // преобразование строки с заданным разделителем в массив

slice

const arr2 = arr1.slice(1, 7); // копировать с 1 до 7 не включая, если не указан второй аргумент, то до конца const arr2 = arr1.slice(1); // копировать с 1 до конца const arr2 = arr.slice(-2); // копировать c 2 элемента с конца и дальше const arr2 = arr.slice(); // копировать весь массив

splice

arr.splice(3, 2); // удалить с 3 индекса 2 элемента arr.splice(2, 0, 'Алина', 'Марина'); // добавление элементов arr.splice(1, 2, 'Алексей", "Билл'); // замена элементов (заменить 2 удаленных элемента на Алексей, Билл) arr.splice(-1, 1); // удалить с последнего индекса-1 1 элемент (отрицательная позиция отсчитывается с конца)

length reverse concat

const a = arr.length; // длина массива (индекс последнего элемента) arr.reverse(); // меняет порядок элементов в массиве на обратный const arr = arr1.concat(arr2, arr3); // создаёт новый массив, в который копируются элементы из массивов

sort

arr.sort(); // сортировка по алфавиту, преобразуя элементы к строке // свой порядок сортировки (сортировка чисел) const arr = [1, 12, 2]; arr.sort((a, b) => { if (a > b) return 1; if (a < b) return -1; }); console.log(arr); // 1, 2, 15

isArray

// проверка на массив const a = Array.isArray([1,2,3]); // true const b = Array.isArray("not array"); // false
