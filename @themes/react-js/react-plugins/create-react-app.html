Create React App

Документация Getting Started Create React App Available Scripts
  • Create React App - создание одностраничных приложений, предоставляет команды для сборки фронтенда
  • Будут установлены пакеты react и react-dom
  • «Под капотом» используются Babel и Webpack, eslint и т.д.
npx create-react-app my-app # React npx create-react-app my-app --template typescript # React + TS npx create-react-app AppName # стандартная установка npx create-react-app AppName --typescript # установка с TypeScript npm run start # старт сервера npm run build # создание оптимизированной сборки приложения в папке build
