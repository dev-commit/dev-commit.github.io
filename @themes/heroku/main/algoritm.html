Алгоритм

1. Регистрация на Heroku

  • https://heroku.com/

2. Скачать Heroku CLI

  • https://devcenter.heroku.com/articles/heroku-cli
  • Download and install -> Windows
  • Скачать и установить

3. Консоль

heroku --help # проверка корректной установки heroku login # залогиниться в CLI
  • Открывается браузер -> Log In -> Logged In (Надпись)
heroku create # создать новый проект
  • В консоли появляются ссылки
  • https://sleepy-lake-01897.herokuapp.com/
  • https://git.heroku.com/sleepy-lake-01897.git

4. Перейти на Heroku Dashboard

  • https://dashboard.heroku.com/apps
  • Зайти в проект -> Deploy

5. Создать git-репозиторий

  • Создать .gitignore
  • sleepy-lake-01897 - название приложения
git init git add . git commit -m 'starter' heroku git:remote -a sleepy-lake-01897 git push heroku master

6. Открыть проект

  • https://sleepy-lake-01897.herokuapp.com/
heroku open # открыть проект по ссылке
