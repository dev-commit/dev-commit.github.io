Jira

1. TODO # Assign to me 2. IN PROGRESS # Start Progress 3. REVIEW # Завершена, на ревью 4. READY FOR QA # После ревью и мерджа в develop 5. IN QA # На тестировании 6. IN DEV # На dev-стенде 7. READY FOR STAGE # Готова к заливке на stage-стенд 8. ON STAGE # На stage-стенде 9. DONE # Закрыта

Жизненный цикл (workflow) задачи

  • OPEN - создание задачи (PM)
  • IN PROGRESS - назначение задачи
  • RESOLVED - завершение задачи
  • REOPENED - открытие задачи заново (PM)
  • CLOSED - закрытие задачи (PM)

Доска

  • BACKLOG - все задачи без оценки
  • TODO - после оценки
  • REJECTED - от тестировщика в случае нахождения багов
  • IN PROGRESS - разработчик берет в разработку
  • REVIEW / TESTING / TO VERIFY - разработчик закончил
  • DONE - переводит тестировщик
